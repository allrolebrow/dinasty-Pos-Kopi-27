# ☕ Dynasty Pos Kopi 27 — Sistem Kasir

> **Dari Ngopi Jadi Story** — Sistem kasir modern berbasis web, gratis, bisa print struk, dan otomatis sync ke Google Sheets.

## 🚀 Demo Langsung
Setelah upload ke GitHub Pages, akses di:
`https://USERNAME.github.io/dynasty-pos/`

## ✨ Fitur Lengkap
- 📋 **POS Kasir** — Input pesanan cepat dengan tampilan menu lengkap
- 🖨️ **Print Struk** — Cetak struk langsung dari browser
- ☁️ **Google Sheets Sync** — Semua transaksi otomatis masuk spreadsheet
- 📊 **Dashboard** — Omset, profit, transaksi, menu terlaris per hari
- 📦 **Kelola Stok** — Set stok & harga HPP setiap menu
- 💳 **Multi-Payment** — Cash, QRIS, Transfer Bank
- 🎫 **Diskon** — Input diskon dalam persen
- 💰 **Tracking Profit** — Hitung keuntungan otomatis berdasarkan HPP
